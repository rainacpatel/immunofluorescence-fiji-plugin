# Installation
To use, install the Immunofluorescence_Analysis-0.1.0-SNAPSHOT.jar file (in the plugins folder) into Fiji. This can be done by clicking Fiji > Plugins > Install... and selecting the file, simply dragging the file into the interface, or saving the file in Fiji's plugins folder directly. Hit save, and restart Fiji.

# Use
This contains 4 plugins for Immunofluorescence image analysis: Process IF Images, Z-Projection IF, Analyze Intensity, and Combine Analysis Summaries. These can be used, in order, to process and analyze confocal fluorescent microscopy images.

Based on the template found at: https://github.com/imagej/example-legacy-plugin
